# Simon Game #
•The Simon Game is a classic memory-based game where players must repeat a sequence of colors in the correct order. The sequence becomes longer each round, making the game more challenging.
•This project is built using HTML, CSS, and JavaScript and runs directly in the browser.
->Features
•Interactive color buttons.
•Random sequence generation.
•Increasing difficulty with each level
Game over screen when the wrong button is pressed.
•Restart option to play again.
->Technologies Used
•HTML – Structure of the game
•CSS – Styling and layout
•JavaScript – Game logic and interactivity.
